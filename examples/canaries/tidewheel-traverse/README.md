# Tidewheel Traverse

Tidewheel Traverse is the mono-compatible SwanSong hardware canary. Its
foreground causeway follows the player immediately while the ocean background
advances at half distance on a slower cadence. The game uses the SDK's fixed
tile-grid cache, clamped camera, and sprite-animation primitives.

Brace before the marked crossings with A, move with left/right, and reset with
Start. The portable model owns all gameplay outcomes; SwanSong remains the ROM
execution and visual evidence backend.
